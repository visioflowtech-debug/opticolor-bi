# Opticolor BI — Ecosistema de Inteligencia de Datos

Proyecto de inteligencia de datos para OPTI-COLOR #2, C.A. Venezuela.
Desarrollado por VisioFlow.

## Estructura

- **/etl** — Pipeline ETL Python integrado con Gesvision API
- **/portal** — Portal web Next.js para usuarios sin licencia Power BI
- **/sql** — Scripts DDL de la base de datos Azure SQL
- **/docs** — Documentación técnica del proyecto
