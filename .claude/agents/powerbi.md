# Power BI Expert

## 5 Informes

### 1. Resumen Comercial
KPIs: Venta Neta, Total Cobrado, Ticket Promedio, Run Rate, OTIF, Clientes Nuevos

### 2. Eficiencia de Órdenes
KPIs: Órdenes hoy/mes, En proceso, Días entrega, Crítico (>15 días)

### 3. Control de Cartera
KPIs: Facturado, Recaudado, Saldo, % Primer Abono, % Pago Total

### 4. Desempeño Clínico
KPIs: Exámenes, % Crecimiento, % Inasistencia, % Conversión

### 5. Inventario
KPIs: Stock, Capital invertido, Unidades por segmento, UPT

## Segmentos Comerciales

- **Luxury:** Marcas premium (Ray-Ban, Gucci, Prada)
- **Intermedias:** Marcas accesibles importadas
- **Be Diferent:** Marca propia Opticolor
- **Contactos:** Categoría separada

## Cálculos DAX

```
Run Rate = (Venta_Diaria × Días_Mes)
OTIF = Entregado_OnTime / Total_Pedidos * 100
% Conversión = Ventas_Con_Examen / Total_Exámenes * 100
UPT = Total_Unidades / Total_Transacciones
```
