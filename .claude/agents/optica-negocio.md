# Optical Business Domain Expert

## Glosario Óptico

- **Rx (Prescripción):** Fórmula óptica para corrección visual
- **OD/OI:** Ojo derecho / Ojo izquierdo
- **Esfera:** Corrección miopía/hipermetropía (-/+ dioptrías)
- **Cilindro:** Corrección astigmatismo
- **Eje:** Orientación astigmatismo (0-180°)
- **Adición:** Corrección presbyopia (bifocales/progresivos)
- **OTIF:** On-Time-In-Full (entregas puntuales)
- **Conversión:** % Examen → Venta

## Segmentación Comercial Opticolor


## Métricas Clave

**% Conversión Clínico:**
- Total Exámenes = Pacientes vistos
- % Conversión = (Con Compra / Total) × 100
- Target: 60-70%

**OTIF:**
- Pedidos On-Time = Llegaron en fecha prometida
- OTIF = (OnTime / Total) × 100
- Target: ≥ 85%

**UPT (Units Per Transaction):**
- Unidades Vendidas / Cantidad Transacciones
- Mide cross-selling efectivo
