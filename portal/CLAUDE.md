# 🎯 Opticolor BI Portal — Contexto Arquitectónico

## ⭐ FUENTE DE VERDAD ARQUITECTÓNICA

**Repository Base:** https://github.com/arhamkhnz/next-shadcn-admin-dashboard

Este Portal de Opticolor está **inspirado y basado** en la arquitectura del admin template de arhamkhnz. Las decisiones de estructura, componentes y patrones siguen ese template como referencia.

**IMPORTANTE:** No clonar template en monorepo. Las customizaciones específicas a Opticolor (auth, SQL, RLS, RBAC, dashboards) se versionan en nuestro `opticolor-bi/portal/`.

## Stack
- Next.js 16.2.4
- TypeScript 5
- React 19
- NextAuth.js v5 (Credentials provider)
- Tailwind CSS v4
- shadcn/ui components
- Recharts (gráficos)
- mssql (Azure SQL connection)
